Run using command line;

python driver.py <Sudoku board>

where

Sudoku board is a string of 81 characters and each character is a digit between 0 and 9 (inclusive)
0 represents an empty cell.

e.g. 
python driver.py 010020300004005060070000008006900070000100002030048000500006040000800106008000000



Program writes output to file output.txt
